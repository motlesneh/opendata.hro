# OpenData.HRO


## Background

**OpenData.HRO** is a web application for the storage and distribution of data. It is based on [CKAN](https://github.com/okfn/ckan) and some of its extensions, all written and maintained by the [Open Knowledge Foundation](http://okfn.org/).

The project was initiated by municipality of Rostock (Germany), more precisely its land registry office.


## In production

**OpenData.HRO** is running in production on the project initiators' servers: http://www.opendata-hro.de
